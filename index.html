Let's Get Started

Getting Started

  • First, set up the proper folder structure.
  • Create your manifest.json file - this is the brain of your pack!
  • Now, design your custom textures, and place them in the correct directories.
  • Optionally, add a pack_icon.png for your pack’s icon.
  • Finally, test your pack by loading it into Minecraft!

Creating Your Manifest

Example:

{
  "format_version": 2,
  "header": {
    "description": "My first awesome texture pack",
    "name": "My Awesome Pack",
    "uuid": "your-unique-uuid-here",
    "version": [1, 0, 0]
  },
  "modules": [
    {
      "type": "resources",
      "uuid": "another-unique-uuid-here",
      "version": [1, 0, 0]
    }
  ]
}

Adding Textures & Pack Icon

Folder structure example:

resource_pack/
|-- textures/
|   |-- blocks/
|   |-- items/
|   |-- entity/
|-- manifest.json
|-- pack_icon.png

pack_icon.png (512x512 px) shows up as your pack's icon.

Blocks

Blocks are the heart of Minecraft! To add or modify block textures, place your custom PNG files in the textures/blocks/ directory of your resource pack.

resource_pack/
|-- textures/
|   |-- blocks/
|       |-- stone.png
|       |-- dirt.png
|       |-- grass_block.png
  • Each PNG file should be named after the block it replaces (e.g., stone.png for the stone block).
  • Recommended size: 16x16, 32x32, or 64x64 pixels (must be square).
  • Test your blocks in-game to ensure they look just right!

Animated Blocks

This guide will help you create animated blocks in Minecraft Bedrock Edition using a resource pack.

  • Step 1: Gather Required Tools
  • Image Editing Software: isopix pro, pixelstudio, or ibispaintx.
  • Text Editor: quick edit, acode, or zarchiver text editor.
  • Step 2: Create Your Animation Image
  • Open Image Editing Software: Start a new project.
  • put the width as the texture of ur block and the height will be "width × no. of frames you want" for example my texture is 16x16 and i want the animated texture to show 4 frames then the width with be 16 but the height will be 16x4 which is 64
  • Design Your Animation: Stack frames vertically in one image (e.g., variations of a diamond block).
  • Save the Image: Save as PNG (e.g., diamond_block.png).
  • Step 3: Create the Flipbook Textures JSON File
  • Open Text Editor: Create a new file.
  • Write JSON Code: Use this template:
    •    [
     {
       "flipbook_texture": "textures/blocks/diamond_block",
       "atlas_tile": "diamond_block",
       "ticks_per_frame": 1,
       "frames": [0, 0, 0, 1, 1, 2, 2, 3]
     }
   ]

  • save it as flipbook_textures.json
resource_pack/
|-- textures/
|   |-- blocks/
|       |-- animated_block.png  (vertical strip with frames)

Tip: Use existing animated blocks (like water.png or lava.png) as references!

Items

Customize your tools, weapons, and more! Place your item textures in textures/items/. 

resource_pack/
|-- textures/
|   |-- items/
|       |-- diamond_sword.png
|       |-- apple.png
  • Each PNG should match the item’s name (e.g., diamond_sword.png).
  • Recommended size: 16x16, 32x32, or 64x64 pixels.
  • Test your items in-game to see your changes!

Particles

Particles add flair to your pack! Edit or add particle textures in textures/particle/. 

resource_pack/
|-- textures/
|   |-- particle/
|       |-- particles.png
  • Each particle effect references a region in particles.png.
  • Use a pixel art editor to design unique effects (sparkles, smoke, etc.).
  • Test in-game by triggering effects (e.g., breaking blocks, fireworks).

Tip: Try the Random Sprite Generator for quick particle ideas!
