Installation

Downloading the Pack

Download the .mcpack file from the official Packaverse website.

Importing to Minecraft

Open the downloaded .mcpack file. Minecraft: Bedrock Edition should automatically open and import the resource pack.

Basic Usage

Activating the Pack

In Minecraft, go to Settings > Global Resources. Find "Packaverse" in the "My Packs" list and activate it.

Blocks

Custom Ores

Packaverse introduces new textures for ores. The deepslate_ruby_ore now has a distinct, vibrant red glow.

Animated Blocks

The Crafting Table has a subtle, glowing animation to make your base feel more alive.

Items

Netherite Sword

The Netherite Sword has been redesigned with a more menacing, blue-hued blade. The item is located at items/netherite_sword.png.

HUD (Heads-Up Display)

Entities
