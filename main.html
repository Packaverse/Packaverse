Let's Get Started

Getting Started

  • Set up the folder structure
  • Create your manifest.json
  • Add custom textures in proper folders
  • Optionally include pack_icon.png
  • Load into Minecraft for testing

Creating Your Manifest

Example:

{
  "format_version": 2,
  "header": {
    "description": "My first awesome texture pack",
    "name": "My Awesome Pack",
    "uuid": "your-unique-uuid-here",
    "version": [1, 0, 0]
  },
  "modules": [
    {
      "type": "resources",
      "uuid": "another-unique-uuid-here",
      "version": [1, 0, 0]
    }
  ]
}

Textures & Pack Icon

Example structure:

resource_pack/
|-- textures/
|   |-- blocks/
|   |-- items/
|   |-- entity/
|-- manifest.json
|-- pack_icon.png

pack_icon.png should be 512x512 px.

Blocks

Place block textures in textures/blocks/

resource_pack/
|-- textures/
|   |-- blocks/
|       |-- stone.png
|       |-- grass_block.png
  • Use proper block names
  • Recommended size: 16x16, 32x32, or 64x64

Animated Blocks

Steps:

  • Use vertical sprite sheet: width = frame width, height = frame height Ã— number of frames
  • Create flipbook file:
[
  {
    "flipbook_texture": "textures/blocks/diamond_block",
    "atlas_tile": "diamond_block",
    "ticks_per_frame": 1,
    "frames": [0, 0, 0, 1, 1, 2, 2, 3]
  }
]

Save as flipbook_textures.json

Items

Put textures in textures/items/

resource_pack/
|-- textures/
|   |-- items/
|       |-- diamond_sword.png
|       |-- apple.png

Particles

Edit textures/particle/particles.png

resource_pack/
|-- textures/
|   |-- particle/
|       |-- particles.png
